# Travel and Tourism Management System

## Project Description
This project is a web-based Travel and Tourism Management System designed to manage bookings, destinations, and customer details.

## Features
- User Registration & Login
- Tour Package Details
- Booking System
- Admin Panel

## Technologies Used
- HTML
- CSS
- JavaScript
- 1 Node.js Backend JavaScript runtime environment
2 Express.js Web framework for routing and middleware management
3 EJS Templates Server-side dynamic HTML templating engine
4 CSS3 Custom responsive frontend styling
5 bcryptjs Secure password hashing library (salt factor: 10)
6 express-session User session and login state management
7 connect-flash Flash messages for success/error alerts
8 VS Code Integrated Development Environment
9 npm Package and dependency management
